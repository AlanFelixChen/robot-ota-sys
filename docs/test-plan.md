# OTA系统测试计划

## 1. 文档概述

### 1.1 文档目的
定义OTA升级系统的**完整测试策略、测试范围、测试方法和验收标准**，确保系统在功能、性能、安全、兼容性等维度达到生产环境要求。

### 1.2 测试范围

| 测试维度 | 测试对象 | 优先级 |
|---------|---------|--------|
| 功能测试 | 云端服务、设备端、管理后台 | P0 |
| 性能测试 | MQTT并发、下载吞吐、API响应 | P0 |
| 安全测试 | 证书验证、固件签名、传输加密 | P0 |
| 兼容性测试 | 多设备型号、多固件版本 | P1 |
| 稳定性测试 | 长时间运行、异常恢复 | P1 |
| 压力测试 | 极限并发、大文件下载 | P1 |
| 回归测试 | 核心业务流程 | P0 |

### 1.3 测试环境

| 环境 | 配置 | 用途 |
|------|------|------|
| 测试环境A | 云端服务 + 100台测试设备 | 功能测试、回归测试 |
| 测试环境B | 云端服务 + 1000台模拟设备 | 性能测试、压力测试 |
| 测试环境C | 云端服务 + 50台真实设备 | 兼容性测试、稳定性测试 |

#### 3.3.2 数据库性能测试

```sql
-- 测试数据准备
INSERT INTO ota.devices (device_id, device_type, hardware_revision, current_version, status)
SELECT 
    'DEV-' || LPAD(generate_series::text, 8, '0'),
    CASE (random() * 3)::int
        WHEN 0 THEN 'TYPE_A'
        WHEN 1 THEN 'TYPE_B'
        ELSE 'TYPE_C'
    END,
    'REV_1.0',
    '1.0.0',
    'ONLINE'
FROM generate_series(1, 100000);

-- 性能测试查询
EXPLAIN ANALYZE
SELECT d.device_id, d.device_type, d.current_version, 
       ur.status as upgrade_status, ur.started_at
FROM ota.devices d
LEFT JOIN ota.upgrade_records ur ON d.device_id = ur.device_id
WHERE d.device_type = 'TYPE_A'
  AND d.status = 'ONLINE'
  AND ur.started_at > NOW() - INTERVAL '7 days'
ORDER BY ur.started_at DESC
LIMIT 100;
```

!/bin/bash

for version in 1.0.0 1.1.0 2.0.0; do
    dd if=/dev/urandom of=firmware_v${version}.bin bs=1M count=50
    sha256sum firmware_v${version}.bin > firmware_v${version}.sha256
done

2. 生成测试证书
echo "2. Generating test certificates..."
for i in {1..100}; do
    openssl req -new -newkey rsa:2048 -nodes \
        -out device_${i}.csr \
        -keyout device_${i}.key \
        -subj "/CN=DEV-$(printf '%08d' $i)"
done

3. 准备数据库测试数据
echo "3. Preparing database test data..."
psql -h localhost -U ota_app -d ota_test <<EOF
INSERT INTO ota.devices (device_id, device_type, hardware_revision, current_version, status)
SELECT 
    'DEV-' || LPAD(generate_series::text, 8, '0'),
    'Model-A',
    'REV_1.0',
    '1.0.0',
    'ONLINE'
FROM generate_series(1, 1000);
EOF

echo "=== Test Data Preparation Complete ==="
